# vue-online-shop-frontend

> 这是一个 vuejs-express-MongoDB 学习练手的小demo，主要用于学习 express 的基本使用和前后端开发练手

## 启动

``` bash
# 安装依赖
yarn install

# start
yarn start

# build
yanr build

```
